# Metronet
Welcome! We are a small MTR server and we are hiring new builders! We have a fun community, an easy application process and no waiting for approval! No problem if you don’t want to become a builder, we have a fun map to explore!

We have a medium sized modpack and a relatively small map so your potato PC can (maybe) run it. There are spectacular stations from our builders that we would like you to explore, from the amazing Spawn HSR station to the hell that is Moorgate, everything is here!

If you want to just chill in this server, we are an amazing server for that! Just join (and please don’t instantly leave that happens so much here).

**So, join [Metronet](https://discord.gg/ekHKmeTp2k) today!**
